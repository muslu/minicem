# minicem
**minicem**, Python için yazılmış, **küçük ve okunabilir** bir metin formatıdır.   Amacı:  - JSON kadar **gürültülü** olmadan - YAML kadar **karmaşık** olmadan - Config / ayar dosyalarında **elle yazıp okumayı kolaylaştırmak**  Kısaca:   “**Liste, dict, bool, None** gibi temel şeyleri çok kısa ve net bir formatla yazmak.”
